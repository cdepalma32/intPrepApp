# Interview Prep App

## Overview
The **Interview Prep App** is a full-stack web application designed to assist users in preparing effectively for technical interviews. Built using the MERN (MongoDB, Express.js, React.js, Node.js) stack, this application offers interactive resources, quizzes, and practice problems to enhance interview readiness.

---

## Features

- **Interactive Quizzes:** Strengthen your technical fundamentals with tailored quizzes.
- **Practice Problems:** Access coding problems categorized by difficulty and topics.
- **Progress Tracking:** Monitor your progress with intuitive dashboards.
- **Responsive Design:** Optimized user experience across devices.
- **Secure Authentication:** User sign-up, login, and profile management with secure authentication.

---

## Technologies

- **Front-End:** React.js, HTML5, CSS3, JavaScript
- **Back-End:** Node.js, Express.js
- **Database:** MongoDB
- **Version Control:** Git & GitHub

## Installation

### Prerequisites
- Node.js
- npm
- MongoDB

### Setup

1. Clone this repository:
```bash
git clone https://github.com/cdepalma32/intPrepApp.git
```

2. Navigate to project directory:
```bash
cd intPrepApp
```

3. Install dependencies:
```bash
npm install
```

4. Start the application:
```bash
npm start
```

5. Open your browser and visit:
```
http://localhost:3000
```
